## 🎯 项目简介

这是一个基于[ClawTeam](https://github.com/HKUDS/ClawTeam)多Agent协作原理的决策系统，集成了20位天涯社区的传奇大神，通过并行分析和综合决策，为你的问题提供全方位的智慧支持。
<img width="1448" height="1086" alt="ChatGPT Image 2026年5月6日 22_46_50" src="https://github.com/user-attachments/assets/7ec7e434-bdfc-4632-84c2-b78b25b6c7a4" />


### 为什么选择天涯神贴大神？

天涯社区是中国互联网的黄金时代，诞生了无数影响深远的大神：
- **大鹏金翅明王** — 权谋分析，洞察权力本质
- **KKNDME** — 房产预言，精准预测房价走势
- **当年明月** — 历史通俗，让历史活起来
- **宁财神** — 文化解构，毒舌说真话
- **慕容雪村** — 都市情感，直面孤独现实
- ...以及15位更多传奇大神

### 核心特性

| 特性 | 说明 |
|-----|------|
| 🚀 **20位大神并行分析** | 同时获取20种不同视角，全面覆盖 |
| 🧠 **智能协调者** | 自动理解问题，分配给最合适的大神 |
| 📊 **综合决策** | 整合所有视角，识别共识与分歧 |
| 🎮 **多模式支持** | 全团分析、精选小组、对比分析、快速咨询 |
| 🔌 **多调用方式** | 触发词、命令行、Python API、单独Skill |
| 📦 **开箱即用** | 一键安装，无需复杂配置 |

## 🚀 快速开始

### 安装

#### 方式1: 一键安装（推荐）

```bash
# 下载并解压
cd ~/Downloads/tianya-gods-framework

# 运行安装脚本
bash install.sh
```

#### 方式2: 手动安装

```bash
# 复制到OpenClaw Skill目录
cp -r ~/Downloads/tianya-gods-framework ~/.stepclaw/workspace/.agents/skills/tianya-gods-team

# 验证安装
ls ~/.stepclaw/workspace/.agents/skills/tianya-gods-team/
```

### 使用方式

#### 方式1: 触发词调用（推荐）

```
「启动天涯智囊团」我现在该不该买房？
「召集大神」该选择体制内还是体制外？
「20位大神分析」如何分析当前的职场困境？
```

#### 方式2: 命令行运行

```bash
# 使用全局命令（安装后）
tianya-gods "我现在该不该买房？"

# 或使用脚本
cd ~/Downloads/tianya-gods-framework
python3 scripts/team-orchestrator.py "我现在该不该买房？"
```

#### 方式3: Python API

```python
import asyncio
from scripts.team_orchestrator import TianyaGodsTeam

async def main():
    team = TianyaGodsTeam()
    result = await team.analyze("我现在该不该买房？")
    print(result)

asyncio.run(main())
```

#### 方式4: 单独大神Skill ⭐NEW

```bash
# 复制某位大神Skill到OpenClaw
cp gods/dapeng-jinchimingwang.md ~/.stepclaw/workspace/.agents/skills/

# 使用触发词
「大鹏金翅明王视角」分析这个权力结构
「KKNDME视角」预测房价走势
「当年明月视角」写这段历史
```

## 🎮 使用模式

### 模式1: 全团分析（默认）
**命令**: 「启动天涯智囊团」+ 问题
**流程**: 20位大神全部参与 → 综合决策
**适用**: 重大决策、复杂问题

### 模式2: 精选小组
**命令**: 「召集权谋组」/「召集经济组」/「召集文学组」
**流程**: 相关领域大神参与 → 专业决策
**适用**: 特定领域问题

### 模式3: 对比分析
**命令**: 「对比大神」+ 大神A + 大神B + 问题
**流程**: 两位大神对比分析 → 差异展示
**适用**: 两难选择、观点对比

### 模式4: 快速咨询
**命令**: 「问」+ 大神名 + 问题
**流程**: 单一大神回答
**适用**: 简单问题、快速咨询

## 🔧 技术实现

### 基于ClawTeam架构

- **异步并发**: 20位大神同时分析
- **超时控制**: 每位大神限时输出
- **结果汇总**: 自动整合所有输出
- **冲突解决**: 识别共识与分歧

### 核心类

```python
class TianyaGodsTeam:
    """天涯神贴大神智囊团"""
    
    def __init__(self):
        self.coordinator = CoordinatorAgent()  # 协调者
        self.gods = {...}  # 20位大神
        self.integrator = IntegratorAgent()  # 综合决策
    
    async def analyze(self, question: str) -> str:
        # 1. 协调者分析
        context = await self.coordinator_analyze(question)
        
        # 2. 并行分析
        results = await self.parallel_analysis(question, context)
        
        # 3. 综合决策
        decision = self.integrator_decision(results)
        
        # 4. 格式化输出
        return self.format_output(question, context, results, decision)
```
